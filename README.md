# Warchoria

A fully responsive movies and TV-shows web application, The platform allows users to browse and search for movies and TV shows, view detailed movie information with trailers, and explore TV show pages including seasons and episodes, It features a trending section to highlight popular titles. State management is handled efficiently using Redux, ensuring smooth and dynamic user interactions across the app.



# Tools & Libraries

-React.js
-react-router-dom
-TailwindCSS
-Axios
-FontAwesome
-jquery
-swiper
-clsx
-react-helmet

# Features

- Responsive design for desktop and mobile.
- allows users to browse and search for movies.
- movie and tv-shaw listing with images, titles, rating and year.
- view detailed movie information with trailers.
- explore TV show pages including seasons and episodes.
- State management using Redux.
- handle not found page.



# Setup & Installation

# 1. Clone the repository
git clone https://github.com/Marwanaashraf/watchoria.git

# 2. Navigate to the project directory
cd watchoria

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
